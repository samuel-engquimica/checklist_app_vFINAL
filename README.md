# Checklist Digital de Portaria e Logística

Aplicação Flutter para registro e validação de checklists de caminhão e caçamba, com coleta de dados operacionais, marcação de itens e captura de assinaturas. Gera relatórios em PDF com layout fiel ao modelo corporativo, incluindo campos específicos como produto perigoso e tipo de operação (carregamento ou descarregamento). Possui calibrador interno para ajuste de coordenadas e inserção de imagens de apoio entre itens. Indicado para uso em tablets na portaria, com foco em padronização, rastreabilidade e redução de retrabalho.
